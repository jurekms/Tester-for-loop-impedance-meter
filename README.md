 Purpose of the tester

The tester is intended for functional verification of loop impedance / loop resistance meters (Zs meters) under real operating conditions.

The device allows a known series resistance RS to be introduced into the measurement circuit and enables evaluation of the meters response to a controlled change in loop impedance. By comparing the measurement results Z and Z + RS, the repeatability, linearity, and the ability of the meter to detect impedance changes can be assessed.

The tester is designed for routine functional checks performed between laboratory calibrations. It does not replace calibration and does not constitute a metrological reference standard.
